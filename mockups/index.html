weft-ui — phase 0 mockups

A reference UI for driving weft, the execution substrate for agent-driven scientific analysis. Three surfaces over one local server that embeds a single controller per workspace. These static mockups carry the visual language and the load-bearing UX decisions before any code is written.

user & agent are peers plans before effects honest degradation — unknown ≠ zero events, not polling one controller per workspace
